# Refactoriser la nomenclature du code spaghetti

## Format 

Travail individuel

## Mission

"Refactorisez le code du 'spagetthi' vu au début du cours en utilisant la nomenclature **BEM: Block-Element-Modifier**. 

1. Identifiez les composants (ex: product-card)
2. Renommez TOUTES les classes selon BEM
3. Enlever les doublons et le superflu dans le CSS
4. Ajoutez des [variables CSS](https://tim-montmorency.com/timdoc/582-211/css/variables/) pour les valeurs répétées
5. Ajoutez des commentaires de documentation
6. Commitez et pushez sur GitHub
